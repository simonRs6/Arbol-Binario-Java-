# 🌳 Árbol Binario de Búsqueda (Java)

Proyecto final para la asignatura de Estructuras de Datos, donde se implementa un Árbol Binario de Búsqueda (BST) de forma manual.

## 📋 Funcionalidades
- **Insertar:** Agrega nodos manteniendo el orden (menores a la izquierda, mayores a la derecha).
- **Recorrido Inorden:** Muestra los datos ordenados ascendentemente.
- **Búsqueda:** Localiza valores dentro del árbol de manera eficiente.



## 🛠️ Detalles Técnicos
- Implementación manual de la clase `Nodo`.
- Lógica de inserción y búsqueda mediante recursividad.
- Sin uso de librerías externas (Standard Java).
